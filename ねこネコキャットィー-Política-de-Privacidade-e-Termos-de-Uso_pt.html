funcionalidades relevantes.
2. Anlise e Melhoria: Uso de dados anonimizados para aprimorar o desempenho do
Aplicativo.
3. Publicidade: Compartilhamento de dados anonimizados com terceiros para otimizar
a publicidade.
O tratamento dos dados de localizao realizado com base em:
GDPR: Artigo 6, pargrafo 1, alnea a ("consentimento") ou alnea f ("interesse
legtimo").
LGPD: Artigo 7, inciso I ("consentimento") ou inciso IX ("legtimo interesse").
